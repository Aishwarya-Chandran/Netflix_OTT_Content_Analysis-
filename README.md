# Netflix / OTT Content Analysis 


This project performs an in-depth analysis of Netflix’s content library, including Movies and TV Shows.  
The aim is to uncover patterns and trends related to content type, release years, countries, genres, ratings, and duration.  

Through data cleaning, validation, and exploratory data analysis (EDA), this project provides actionable insights into Netflix’s content strategy, audience targeting, and global expansion approach.


## Tech Stack
- **Python** – Data manipulation and analysis 
- **Pandas** – Data cleaning and preprocessing  
- **Matplotlib & Seaborn** – Visualization and EDA  
- **Jupyter Notebook** – Interactive analysis and reporting
